# ✅ f) FAQ WITHOUT DATABASE (DUMMY DATA)

## 📄 File: `FAQServlet.java`

📍 `src/com/example/`

```java
package com.example;

import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;

public class FAQServlet extends HttpServlet {

    protected void doGet(HttpServletRequest req, HttpServletResponse res)
            throws ServletException, IOException {

        PrintWriter out = res.getWriter();

        // Dummy Data
        String[][] faq = {
            {"Java", "What is Java?", "Java is a programming language"},
            {"Servlet", "What is Servlet?", "Servlet is a server-side Java program"},
            {"JSP", "What is JSP?", "Java Server Pages used for web development"}
        };

        out.println("<html><body>");
        out.println("<h2>Dynamic FAQ Page</h2>");

        for (int i = 0; i < faq.length; i++) {
            out.println("<h3>" + faq[i][0] + "</h3>");
            out.println("Q: " + faq[i][1] + "<br>");
            out.println("A: " + faq[i][2] + "<br><br>");
        }

        out.println("</body></html>");
    }
}
```

---

## 📄 `web.xml`

```xml
<servlet>
  <servlet-name>FAQServlet</servlet-name>
  <servlet-class>com.example.FAQServlet</servlet-class>
</servlet>

<servlet-mapping>
  <servlet-name>FAQServlet</servlet-name>
  <url-pattern>/faq</url-pattern>
</servlet-mapping>
```

# ✅ h) SURVEY WITHOUT DATABASE (SESSION BASED)

👉 This is **best alternative** because it still behaves dynamically.

---

## 📄 File: `SurveyServlet.java`

📍 `src/com/example/`

```java
package com.example;

import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;

public class SurveyServlet extends HttpServlet {

    protected void doPost(HttpServletRequest req, HttpServletResponse res)
            throws ServletException, IOException {

        HttpSession session = req.getSession();

        String animal = req.getParameter("animal");

        // Initialize counts if not exists
        Integer dog = (Integer) session.getAttribute("Dog");
        Integer cat = (Integer) session.getAttribute("Cat");
        Integer parrot = (Integer) session.getAttribute("Parrot");
        Integer cow = (Integer) session.getAttribute("Cow");
        Integer none = (Integer) session.getAttribute("None");

        if (dog == null) {
            dog = cat = parrot = cow = none = 0;
        }

        // Increment selected
        if (animal.equals("Dog")) dog++;
        else if (animal.equals("Cat")) cat++;
        else if (animal.equals("Parrot")) parrot++;
        else if (animal.equals("Cow")) cow++;
        else if (animal.equals("None")) none++;

        // Save back
        session.setAttribute("Dog", dog);
        session.setAttribute("Cat", cat);
        session.setAttribute("Parrot", parrot);
        session.setAttribute("Cow", cow);
        session.setAttribute("None", none);

        int total = dog + cat + parrot + cow + none;

        PrintWriter out = res.getWriter();
        out.println("<html><body>");
        out.println("<h2>Survey Result</h2>");

        if (total > 0) {
            out.println("Dog: " + (dog * 100 / total) + "%<br>");
            out.println("Cat: " + (cat * 100 / total) + "%<br>");
            out.println("Parrot: " + (parrot * 100 / total) + "%<br>");
            out.println("Cow: " + (cow * 100 / total) + "%<br>");
            out.println("None: " + (none * 100 / total) + "%<br>");
        }

        out.println("Total Votes: " + total);

        out.println("</body></html>");
    }
}
```

---

## 📄 `web.xml`

```xml
<servlet>
  <servlet-name>SurveyServlet</servlet-name>
  <servlet-class>com.example.SurveyServlet</servlet-class>
</servlet>

<servlet-mapping>
  <servlet-name>SurveyServlet</servlet-name>
  <url-pattern>/survey</url-pattern>
</servlet-mapping>
```
