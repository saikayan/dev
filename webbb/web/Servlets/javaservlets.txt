# 🔧 COMMON PROJECT STRUCTURE (Same for all)

Create **Dynamic Web Project** in Eclipse:

```
ProjectName/
│
├── src/                → Java files (Servlets)
│   └── com/example/
│       └── *.java
│
├── WebContent/ (or webapp/)
│   ├── *.html
│   └── WEB-INF/
│       └── web.xml
```

# ✅ a) HELLO WORLD – GENERIC SERVLET

### 📄 File: `HelloServlet.java`

📍 Location: `src/com/example/`

```java
package com.example;

import java.io.*;
import javax.servlet.*;

public class HelloServlet extends GenericServlet {

    public void service(ServletRequest req, ServletResponse res)
            throws ServletException, IOException {

        res.setContentType("text/html");
        PrintWriter out = res.getWriter();

        out.println("<html><body>");
        out.println("<h1>HELLO WORLD</h1>");
        out.println("</body></html>");
    }
}
```

---

### 📄 File: `web.xml`

📍 Location: `WebContent/WEB-INF/`

```xml
<web-app>

  <servlet>
    <servlet-name>HelloServlet</servlet-name>
    <servlet-class>com.example.HelloServlet</servlet-class>
  </servlet>

  <servlet-mapping>
    <servlet-name>HelloServlet</servlet-name>
    <url-pattern>/hello</url-pattern>
  </servlet-mapping>

</web-app>
```

👉 Run:
`http://localhost:8080/ProjectName/hello`

---

### 📄 File: `EmpServlet.java`

```java
package com.example;

import java.io.*;
import java.util.*;
import javax.servlet.*;

public class EmpServlet extends GenericServlet {

    public void service(ServletRequest req, ServletResponse res)
            throws ServletException, IOException {

        res.setContentType("text/html");
        PrintWriter out = res.getWriter();

        Enumeration<String> paramNames = req.getParameterNames();

        out.println("<html><body>");

        while (paramNames.hasMoreElements()) {
            String name = paramNames.nextElement();
            String value = req.getParameter(name);
            out.println(name + " : " + value + "<br>");
        }

        out.println("</body></html>");
    }
}
```

</servlet>

  <servlet-name>EmpServlet</servlet-name>
  <url-pattern>/emp</url-pattern>
</servlet-mapping>
```

<option>Blue</option>
</select>
<input type="submit">
</form>
</body>
</html>
```

---

### 📄 File: `ColorServlet.java`

```java
package com.example;

import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;

public class ColorServlet extends HttpServlet {

    protected void doGet(HttpServletRequest req, HttpServletResponse res)
            throws ServletException, IOException {

        String color = req.getParameter("color");

        PrintWriter out = res.getWriter();
        out.println("<html><body>");
        out.println("Selected Color: " + color);
        out.println("</body></html>");
    }
}
```

---

### 📄 `web.xml`

```xml
<servlet>
  <servlet-name>ColorServlet</servlet-name>
  <servlet-class>com.example.ColorServlet</servlet-class>
</servlet>

<servlet-mapping>
  <servlet-name>ColorServlet</servlet-name>
  <url-pattern>/color</url-pattern>
</servlet-mapping>
```
